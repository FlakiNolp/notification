1.2 Краткое описание возможностей
          Программа позволяет владельцам сайтов или других информационных систем, где требуется система логгирования и оповещения, отправлять http post запросы с телом в форматjson, включающим “code”, “message”, “additional” на наш сервер для обработки, оповещения, сохранения и дальнейшего предоставления этой информации пользователю в нужное время по средствам http get запросов с параметрами для возврата нужной информации в виде json.
          Оповещение происходит в момент отправки post запроса. Значения информационных запросов отправляется на сервисы по выбору пользователя в настройках, раздел <Основные> (E-mail, Вконтакте, Telegram, webhook).
          Также, в настройках, раздел <Отслеживание> имеется возможность мониторинга количество уведомлений за определённые временные промежутки.
